Mi Librería de Alertas

Una librería de alertas moderna y personalizable

Ejemplo de uso:

// Alerta simple
CustomAlert.fire({
  type: 'success',
  title: '¡Éxito!',
  text: 'Operación completada correctamente'
});

// Confirmación
CustomAlert.confirm({
  title: '¿Estás seguro?',
  text: 'Esta acción no se puede deshacer'
}).then(result => {
  if (result) console.log('Confirmado');
});
